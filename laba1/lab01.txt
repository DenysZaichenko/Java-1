java

Написати консольний застосунок (jar файл, який можна запустити командою java -jar app.jar), який при запуску виводить дерево папок і файлів від поточної директорії вглибину.
У вигляді

current folder
	folder1
		file1
		file2
		file3
	folder2
		folder3
			file4
		file5
		....


При роботі використовувати java+maven без додаткових фреймворків і бібліотек.
В програмі використати рекурсивні виклики.


Для запуску програми:

java -cp target/laba1-1.0.jar com.example.App
